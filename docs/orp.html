ORP Measurement

# measureORP

Starts an ORP measurement. ISE_ORP::ORP and ISE_ORP::Eh are updated.

Usage

  • C++
  • Python
  • Shell
ISE_pH::measureORP();

Example

  • C++
  • Python
#include <ISE_ORP.h>
ISE_ORP orp;

orp.measureORP()

Related

  • measuremV

Returns

the measured result in mV, or -1 on error

ALSO...

The class variable Eh, is also updated.

# setProbePotential

Saves the connected probe's potential in EEPROM so that Eh can be calculated. After setting this value, the class variable Eh is automatically updated with each call to measureORP.

Usage

  • C++
  • Python
ISE_ORP::setProbePotential(195);

Example

  • C++
  • Python
#include <ISE_ORP.h>
ISE_ORP orp;

orp.setProbePotential(195)

Related

  • getProbePotential

Returns

none
